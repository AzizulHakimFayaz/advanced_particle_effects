# Changelog

## [1.0.0] - 2025-01-XX

### Added
- Initial release
- NetworkedParticleSystem widget with full customization
- User-controllable particle count, size, colors, line width, speed
- Smart defaults - works with zero configuration
- Connection lines with distance-based opacity
- Bounce behavior
- 8+ preset particle effects (fireworks, snow, rain, confetti, etc.)
- Advanced physics (gravity, wind, magnetic fields)
- Multiple particle shapes (circle, square, triangle, star, heart, diamond, line)
- Blend mode support
- Rainbow color mode
- Interactive examples
